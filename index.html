Jira app for quote work

Quotes inside Jira.

Quotaurus helps Jira Cloud teams create quotes, link them to tickets, generate PDF attachments, send configured quote emails, track status and version history, and automate recurring monthly quote drafts.

Contact Quotaurus Get support

Why it exists

The win is keeping quote context where Jira work happens.

The app is strongest when quotes, Jira tickets, PDF outputs, client details, and recurring monthly work would otherwise drift apart. Quotaurus keeps those records in Jira project and issue views.

High-win scenarios

What people will actually look for.

01

Create quotes from Jira work

Start a draft quote from a Jira issue panel, or create it from the project page and keep the quote tied to the project scope.

02

Link quotes to tickets

Link existing open quotes to issues, view linked quote references from the issue panel, and open the quote back in the project page.

03

Price with real line items

Use clients, products, rates, quantities, VAT, currency, revenue streams, systems, PO numbers, and forecasted completion dates.

04

Generate PDFs and send quotes

Preview a quote PDF and send it by email when project Mailgun settings are configured. Sent quotes store recipient and sent-date details.

05

Track status and history

Work with Draft, Sent, Accepted, Rejected, Invoiceable, Closed, and Cancelled states. Quote updates keep version history for later reference.

06

Handle recurring quote work

Configure recurring quote templates that generate monthly drafts on the selected billing day, including last-day-of-month schedules.

Flow

A practical path through the current app.

  1. Create Create from a ticket or from the project Quotes page.
  2. Price Add client, products, rates, quantities, VAT, and currency.
  3. Link Attach open quotes to the Jira issues they cover.
  4. Send Generate a PDF and send email when Mailgun is configured.
  5. Track Follow status, sent metadata, linked tickets, and versions.
  6. Repeat Use recurring templates and reports for monthly quote work.
