What to expect

  • Installation requires Jira Cloud admin permission.
  • Billing is expected to run through Atlassian Marketplace.
  • Any free trial, user tier, or paid plan should be confirmed on the listing.

Before installing

  • Review the app permissions on the Atlassian install screen.
  • Check the Security & Trust page for current data-flow notes.
  • Confirm whether your team wants Mailgun email sending configured.

What is included in the app workflow

Quote creation from Jira, issue linking, structured line items, PDF generation, Mailgun-backed quote email sending when configured, quote status tracking, version history, and recurring monthly draft templates.

Questions to ask

If your team needs procurement details before the Marketplace listing is visible, email Quotaurus with your Jira Cloud site size, expected use case, and any security or data review requirements.
